# Talantongo Marketing Services Website

A modern, beautiful website built with Astro featuring a sunset color scheme and sunrise-inspired logo.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:4321`

### Build

Build the site for production:

```bash
npm run build
```

### Preview

Preview the production build:

```bash
npm run preview
```

## Features

- **Sunset Color Scheme**: Beautiful yellow to orange gradient throughout
- **Sunrise Logo**: Custom logo with gradient from yellow/orange (bottom-left) to darker orange (top-right) with bold black "T"
- **Responsive Design**: Works beautifully on all devices
- **Modern UI**: Clean, professional design with smooth animations
- **All Content Sections**: Hero, About, Services, Philosophy, Values, Who We Serve, and What Makes Us Different
